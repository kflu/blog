自由泳

Jul 7, 2017

最近两个月自由泳的训练还是很有成效的！现在可以1小时内游2km。两边换气、early catch，姿势不错， 感觉游的效率比较高。这两次开始练转身，今天已经能比较稳定的转了。加油！自由泳对肩膀还是有负担 的，游下来肩膀还是有点反应，要注意。

另外一个好处是，肩宽了，胸大了，胳膊粗了，嗯。
